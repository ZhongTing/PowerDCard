
## Introduction
這是一個chrome的擴充套件  
讓使用者可以更方便的瀏覽 [Dcard](https://www.dcard.tw/)

**這個擴充套件，在某一次Dcard大改版後就不在適用了**  
**不過套件的功能後續有被整合進去現在的Dcard**

## Functions
* 快速回應
* Anchor hyperlinks
* 網址自動超連結
* 預覽文章內圖片
* 文章嵌入Youtube短片
* 隱藏話梗
* 校園聊天搜尋
